Widgets are script files that enhance BAR. Most of them are only useful in NuttyB.

These need "Allow Custom Widgets" enabled in lobby settings which is the default for us.

## Must-Have

- [**"New" Raptor Panel**](https://github.com/goldjee/BAR-Widgets/) - Detailed damage and aggro info [discord](https://discord.com/channels/549281623154229250/1429215779311587380)
- [**Map Start Lines Widget**](https://github.com/timuela/BAR-auto_draw_grid) - Layout planning lines [discord](https://discord.com/channels/549281623154229250/1377685542187303002)
- [**Reclaim Selected**](https://github.com/manshanko/bar-widgets) - One-button reclaim (essential) [discord](https://discord.com/channels/549281623154229250/1134843012405534831)
- [**Holo Place**](https://github.com/timuela/BAR-widgets) - Auto-place next building [discord](https://discord.com/channels/549281623154229250/1380625454960083155)
- [**Build Orders**](https://github.com/noryon/BARWidgets/tree/main/BuildOrders) - Replace buildings efficiently
- [**Always Show Building Grid**](https://discord.com/channels/549281623154229250/1168959237641216131/1221702143761842267) - Building grid always shows at your current cursor location, instead of only showing when placing building ghosts.

## Helpful

- [**"Old" Raptor Stats Panel**](https://gist.github.com/tetrisface/2f99f5a5b179e3ac42e7e63825880713) - Shows your aggro levels [discord](https://discord.com/channels/549281623154229250/1203485910512173096)
- [**Layout Planner**](https://github.com/noryon/BARWidgets/tree/main/LayoutPlanner) - Perfect placement assistance [discord](https://discord.com/channels/549281623154229250/1383202304345378826)
- [**Anti Reminder**](https://discord.com/channels/549281623154229250/1206703169585811456) - Shows a blinking red warning text at 66% raptor tech evolution if you don't have any anti nuke.
- [**Queen Ping**](https://github.com/timuela/BAR-widgets/blob/main/queen_ping.lua) - Track queen deaths
- [**Shield Ground Rings**](https://discord.com/channels/549281623154229250/1267570039197007943/1267570039197007943) - This widget adds shield rings on the ground with different colors and animates the queued shields' rings for added visibility. ⚠️Only works on flat map such as Full Metal Plate

## Install Guide

1. Open the `BAR Launcher` and click "Open Install Directory".

![](/widgets-bar-show-data-folder.webp)

2. Create the folders `LuaUI\Widgets\` in that Directory.
3. Start the game if not already started.
4. Use `Settings` -> `Custom` to enable/disable widgets.
-  Use `/luaui reload` in public chat to reload widgets.

## Widgets to Disable

This needs `/widgetselector` / `F11`, it's easy to break the game in there, take care!

- **Disable Reclaim Field Highlight Widget** - Too expensive for raptor games
- **Disable LoS view** - Performance impact
